# Files In This Directory

This directory is a standalone, lightweight Web SDK Dynamic Forms prototype.

## Files

- `README.md`: Quick start, build/run commands, and short rationale.
- `DESIGN.md`: Main assignment-oriented documentation covering requirements, deliverables, architecture, API, extensibility, and testing strategy.
- `FILES.md`: This file.
- `package.json`: Minimal TypeScript build scripts and dev dependency.
- `tsconfig.json`: Strict TypeScript compiler settings for source and sample files.

## Subdirectories

- `src/`: TypeScript-authored SDK/core/renderer prototype.
- `sample/`: Browser sample app that renders a draggable weblayer-style form.
- `example/`: Example payloads that mimic what a delivery endpoint could return.
- `dist/`: Generated JavaScript output from `npm run build`; do not edit manually.
